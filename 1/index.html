11. Linux для Начинающих - Архивирование и Сжатие tar, gzip, bzip2, xz, zip

 

tar cf  mytar.tar  Folder1   - заархивировать Folder1 (tar)

tar xf mytar.tar  - разархивировать архив (tar)

gzip     / bzip2     / xz      – скомпрессировать файл (три метода)

gunzip /  bunzip2 / unxz  – раскомпресировать файл (три метода)

 

tar cvjf myBZIP2.bz2  Folder1    – сжать Folder1 (tar) архиватором bzip2 (лучше чем gzip)

ключи и расширение выбора метода архивирования «tar cvjf …»   -

      z — gzip (gz),  j — bzip2 (bz2),  J — xz (xz) (на Ubuntu нужен xz-utils, на CentOS - xz)

tar xvf  myBZIP2.bz2   - распаковать архив (x — распаковать, v — на экран, f - файл)

tar tf myBZIP2.bz2    - посмотреть что внутри архива

 

zip –r myZIP.zip Folder1   - Запаковать Folder1 в ZIP

unzip myZIP.zip                - Распаковать файл myZIP.zip

16. Linux для Начинающих - Права Доступа и владения файлами и директориями

 

chown   новыйВладел. файл/дир   –   изменить владельца файла/директории

chgrp   новаяГруппа. файл/дир   –   изменить группу файла / директории

сhmod – изменить права доступа на файл / директорию

 

chmod    ugo+x (a+x)   myfile.txt    -    права доступа - довавить X всем

сhmod  g-rw   myfile.txt   -    права доступа - убрать RW у группы

chmod  o=rw   myfile.txt   -    права доступа - установить RW всем остальным

 u = user

 g = group

 o = other

 a = ugo

 

chmod  777   myfile.txt   -    права доступа - установить RWX всем

chmod  741   myfile.txt    -    права доступа - установить:   RWX   владельцу, R - -    группе,                                                                                            - - X   всем остальным

r = 4

w = 2

x = 1

 

chmod  1777 myDir  (chmod o+t myDir)    - права доступа - включить StickyBit

chmod 0777 myDir  (chmod o-t myDir)    - права доступа - выключить StickyBit
